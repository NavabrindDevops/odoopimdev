Odoo Import Product and Product Variant App

Odoo import product variant app helps to easily import product and product's variant with all fields such as product image, price, quantity, stock, category, description, UOM, taxes, variants attributes and attributes values etc from csv or excel. also you are able to import custom fields of product variants from the csv/xls file. this app also provides the option to import and update product data from excel or csv files.
